#########################
# Parallel Bitonic Sort #
#########################

Setup Instructions
------------------
```
$ chmod +x setup.sh
$ ./setup.sh
```

How to execute Bitonic Sort program
-----------------------------------
```
$ ./bin/bitonic.o <INPUT_DATA_FILE_PATH> <NUMBER_OF_THREADS>
```

How to run 19 trials of Bitonic Sort
------------------------------------
```
$ chmod +x run.sh
$ ./run.sh <INPUT_DATA_FILE_PATH> <NUMBER_OF_THREADS> <OUTPUT_FILE_PATH>
$ cat <OUTPUT_FILE_PATH>
```
